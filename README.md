# docker-mvn-node-playwright
Docker: Maven, Node.js, Playwright, Java 21

## Description
Docker image for Java based automated UI tests with Playwright.

## Contained Libs
* openjdk 21
* Maven 3.9
* Playwright and Browser Binaries (path: /opt/ms-playwright)
